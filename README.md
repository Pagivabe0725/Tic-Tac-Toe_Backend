# Tic Tac Toe – Backend

Backend service for a full-stack Tic Tac Toe application, handling authentication, game logic, and persistent data storage.

> Focused on clean architecture, modular design, and scalable game logic.

## 🚀 Live Application

https://tic-tac-toe-2025-mu.vercel.app/tic-tac-toe

## 🧰 Tech Stack

- Node.js, Express
- GraphQL API
- MySQL (Sequelize ORM)
- Session-based authentication
- CSRF protection

## ✨ Responsibilities

- User authentication and session management
- Game logic and AI (multiple difficulty levels)
- Handling opponent moves
- Persistent game state storage
- GraphQL layer for flexible data querying
